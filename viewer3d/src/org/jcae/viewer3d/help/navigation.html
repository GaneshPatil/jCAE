3D navigation help

Rotating

  • Move the mouse with the main (left) mouse button pressed. The rotation is in the direction of the mouse movement, with a default rotation of 0.01 radians for each pixel of mouse movement.

Zooming

  • Move the mouse with the middle mouse button pressed. Use Alt+main mouse button on systems without a middle mouse button. If the "Alt+main mouse button" action is already reserved by your system (i.e. Many Unix windows manager use it to move a window), use "Alt+Shift+main mouse button".
  • If you zoom a lot, you may jump inside the object you are looking at. This is a good way to see the interior of an object.

Panning

  • Move the mouse with the right mouse button pressed.
