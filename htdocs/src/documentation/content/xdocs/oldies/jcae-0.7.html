Installation

To run jCAE 0.7 you need install the 3 following software:

Name URL Notes
Opencascade 5.2 http://www.opencascade.org You only need to install the binary files.
Java Runtime Environement 1.4.2 http://java.sun.com A JDK is also fine.
Java 3D http://java.sun.com/products/java-media/3D/ Ensure you install it in the JRE directory you will use to run jCAE

Then unzip jcae-xx.zip and run jcae-xx/bin/runide.sh on Linux or jcae-xx/bin/runidew.exe on Windows.

Here is the list of dependencies included in the distribution package:

Name URL Notes
Netbeans Platform 3.6 http://www.netbeans.org/ Nearly all GUI features
JGoodies Looks 1.2.2 https://looks.dev.java.net/ Give jcae a better look
Syn 3D 0.9 http://jsynoptic.sourceforge.net/ Help to display 3D
GNU Trove 1.0.2 http://trove4j.sourceforge.net/ Make the mesher faster
Log4j 1.2.8 http://logging.apache.org/log4j/docs/ Help to debug and maintain the mesher
Groovy 1.0b6 http://groovy.codehaus.org/ Will help to easily create modules for jCAE (this is currently not activated).
